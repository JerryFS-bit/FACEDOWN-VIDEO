# FaceDown-Video
Descarga videos de Facebook en calidad HD / SD muy fácilmente.

* instalación en Linux
~~~
git clone https://github.com/JerryFS-bit/FACEDOWN-VIDEO.git
cd FACEDOWN-VIDEO
pip install -r requirements.txt
python3 FaceDownVideo.py
~~~
